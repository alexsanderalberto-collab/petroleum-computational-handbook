# Petroleum Computational Handbook

Repositório dedicado à integração entre engenharia de petróleo, petrofísica e computação científica utilizando Python.

O objetivo do projeto é transformar conceitos clássicos da engenharia de reservatórios em workflows computacionais progressivos, documentando estudos, revisões técnicas e aplicações práticas.

## Áreas abordadas
- Petrofísica
- Engenharia de reservatórios
- Propriedades de fluidos
- Modelagem computacional
- Automação de cálculos de engenharia
- Python aplicado à subsuperfície

## Estrutura atual

### 01_fundamentos_petrofisica
- Porosidade
- Saturação
- Volume poroso
- Cálculos básicos de reservatório
- Introdução ao Python científico

Construir um acervo computacional contínuo baseado em referências clássicas da indústria, como:
